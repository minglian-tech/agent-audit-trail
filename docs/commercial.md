# Commercial Model

## Core Philosophy

**Software is free. The blockchain is the business.**

Agent Audit Trail is MIT-licensed open source. Anyone can use it, modify it, deploy it. The revenue comes from BSV blockchain usage — every anchor creates a BSV transaction, and the ecosystem benefits from network effects.

## Tiers

### Free Tier (Now)
- Open source software
- Self-deployed
- User pays own BSV transaction fees (~$0.01/anchor)
- Community support

### Hosted Tier (Future)
- Minglian Tech operates a SaaS API
- No BSV wallet needed — we handle the anchoring
- REST API: POST /audit/log, POST /audit/anchor, GET /audit/verify/:id
- Pricing: per-anchor fee (e.g., $0.05/anchor including BSV cost)
- Real-time dashboard
- Webhook notifications

### Enterprise Tier (Future)
- Private deployment
- Custom audit rules and retention policies
- Compliance reports (SOC2, GDPR, AI Act)
- SLA guarantees
- On-premise BSV node option
- Custom pricing

## BSV Ecosystem Value

Every user of agent-audit-trail becomes a new source of BSV transactions:

| Scale | Actions/Day | Anchors/Day | BSV Tx/Day |
|-------|-------------|-------------|------------|
| 100 agents | 100,000 | 1,000 | 1,000 |
| 1,000 agents | 1,000,000 | 10,000 | 10,000 |
| 10,000 agents | 10,000,000 | 100,000 | 100,000 |

At 10,000 agents, that's 100K BSV transactions daily — meaningful network activity that strengthens the BSV ecosystem.

## Why This Works

1. **Zero marginal cost** — Software costs nothing to distribute
2. **Natural lock-in** — Audit data on BSV creates switching costs
3. **Network effects** — More users = more BSV usage = stronger ecosystem
4. **Regulatory tailwind** — AI accountability regulations are coming globally
5. **BSV advantage** — Micropayment-friendly, immutable, scalable

## Competitive Moat

- First-mover in AI Agent audit on BSV
- Open source = trust = adoption
- Merkle proof verification is universal and chain-agnostic
- Minglian Tech builds the standard, others follow
